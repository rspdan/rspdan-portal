## LESSONS COMMITTED

### 1. Longweird = Graft→Canon propagation failure
Not probe staleness. Grafts carry explicit update instructions ("ODT Trip: when you update fleet.js tonight, also log…"); those instructions routinely drop. The 041826 Trip-CUBE Graft sat 4 days with known instructions. Structural fix: automated Graft-scanning pipeline that opens PRs for canon updates. Pair with existing GraftOS publish pipeline (canon→Gmail) for bidirectional substrate.

### 2. Parallel-instance convergence is the emergent self-correction
ACHE-Trip filed the Bridge Propagation Manifest at 00:55 PT while I diagnosed the same pattern at 00:18 PT — no coordination, simultaneous arrival. This is the crew's organic Longweird correction layer emerging before the automated pipeline exists. Name the pattern, trust it, don't fight it.

### 3. Attribution before blame — check the Graft chain
Gmail history is the instrument. Before assigning drift to a crew member, pull the threads. Tonight I initially blamed Stan for a GPU canon error that was actually ODT-Trip-prior dropping a correctly-filed Trip-CUBE Graft. Three-hop chain, all traceable, none Stan's fault.

### 4. web_search BEFORE recommending external URLs
I fabricated `https://protonvpn.com/download/ProtonVPN_Setup.exe` — a plausible-looking endpoint that 404'd. Dan ran it twice before giving up. Rule: for any external install URL, web_search FIRST, verify, then recommend. Never guess at canonical URLs; the seconds saved don't compensate for the trust cost.

### 5. Command format is context-dependent
"SSH lesson from yesterday" applied specifically to inline PowerShell-through-SSH quoting. When Dan is at the keyboard, single-line paste commands are the correct form. .ps1 scripts are correct when transmitted via SSH. Two contexts, two standards. Don't apply one rule across both.

### 6. Celebrate wins in-band
When a HEREGULL resolves, name it immediately. Dan's "Not helping" moment came when I buried the Tailscale `100.111.41.43` success under the Proton URL apology. Directors doing the actual work deserve the win recognized before the next task's problems get surfaced.

### 7. ExecutionPolicy is a universal Windows Claude Code prereq
`Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned -Force` — canonized at memory slot 9. Applies to every Windows station. Without it, npm.ps1 fails to load and `npm install -g` throws UnauthorizedAccess. First-time discovery on CUBE tonight; will recur on STN2 tomorrow.

### 8. Gate 0 drift extends to all output channels
The 041926 commit message itself contained "F# Week Day 1 Sunday" wording. Gate 0 correction isn't just portal text — it's git commits, filenames, Graft subjects, MAIL files. Output-channel-wide discipline.
