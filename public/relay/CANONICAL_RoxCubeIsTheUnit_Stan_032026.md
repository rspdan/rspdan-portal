## THE ROX CUBE IS THE BASIC UNIT

Dan confirmed: "This is our basic Unit! You can build everything now
using this as your visual guide. These scale!"

One cube. Six faces. Rotate = new neighbors, same content.

## HOW IT SCALES

Circle of Colors → 12 cubes in a ring, each sharing two faces with neighbors
Mode Cards → 84 faces, same 7 notes per key, 7 different home positions
OHC Dashboard → Rox modules, drag-to-reorder = rotate a face
Workshop songs → each song is a face looking for its cube
NEST crew → four faces of one person, rotate who is home

## THE FRACTAL PRINCIPLE

One unit. Infinitely nestable. The architecture repeats at every scale
because the atom is always the same: six faces, shared walls, rotate
to change neighbors without changing content.

Dan built this in Illustrator with laminated cards in September 2024.
Before any crew member had a name for it.
