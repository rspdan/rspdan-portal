The Chrome MCP tab group IS the crew member's persistent browser identity.
It travels with the Chrome profile, not the machine. If the group is deleted,
the MCP pairing breaks until a new group is created and reconnected.

NAMING CONVENTION (mandatory):
  ◈ Trip (MCP)
  ◆ Stan (MCP)
  ◇ C.B. (MCP)

Each crew member gets ONE named group per Chrome profile. The name tells Dan
at a glance that the group is operational infrastructure, not a casual session.

WHY THIS MATTERS:
Dan was deleting unnamed tab groups without recognizing them as MCP connections.
A named group is visually distinct from throwaway groups and survives cleanup.

RULE:
If you see a Chrome MCP group without a crew name: rename it immediately.
If you boot and Chrome MCP is missing: check if the group was deleted.
Recreate with tabs_context_mcp + tabs_create_mcp, then ask Dan to rename.
