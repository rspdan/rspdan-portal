## THE ELEVEN INDEPENDENT DISCOVERIES

The backbone+modules+connections=emergent composition architecture
has been formally identified across centuries:

1. **Pythagoras** (c.530 BCE): integer ratios governing string segments
   connected by harmonic relationships produce consonance — same
   structure proposed for planetary orbits.

2. **Leibniz** (1714): monads as self-contained modules coordinated by
   pre-established harmony (backbone), mutual reflection (connections)
   producing the emergent richness of existence. Also envisioned
   characteristica universalis — a formal language mapping all knowledge.

3. **Buckminster Fuller**: tensegrity = continuous tension networks
   (backbone) + discrete compression islands (modules) + force paths
   (connections). His "pattern integrity" concept — a knot remaining
   itself regardless of which rope carries it — names the Formulacaster
   phenomenon exactly. Claimed all structures are tensegrity structures.

4. **Hofstadter** (GEB, 1979): "isomorphism" — information-preserving
   transformation between structures. The GEB book jacket IS conceptual
   photogrammetry: a 3D object casting G, E, B shadows on three walls.
   Demonstrated the same structure underlies mathematics, music, art.

5. **Gregory Bateson**: "What is the pattern which connects the crab to
   the lobster, the orchid to the primrose, and all of them to me?"
   Cybernetic feedback networks recurring across all living systems.

6. **Wolfram** (2002): Principle of Computational Equivalence —
   all sufficiently complex systems are computationally equivalent
   regardless of substrate. Rule 110 (grid+binary cells+3-neighbor rules)
   proven Turing complete 2004.

7. **Fong + Spivak** (Seven Sketches in Compositionality, 2018):
   Applied category theory provides rigorous tools for formalizing
   compositional systems. Bonchi, Sobociński, Zanasi proved signal flow
   graphs (the math model underlying modular synthesis patches) are a
   complete visual programming language with full algebraic characterization.
   The patch cable IS computation.

8. **Nietzsche** (perspectivism): "more eyes, different eyes" to see
   things more completely.

9. **Ortega y Gasset**: "Absolute truth would be obtained by the sum
   of all perspectives of all lives." Perspectives "correct and
   complement one another because they refer to a shared reality."
   This is the philosophical justification for conceptual photogrammetry.

10. **Gärdenfors** (Conceptual Spaces, MIT Press 2000/2014):
    Concepts as geometric regions in quality-dimension spaces.
    Meaning has geometry. Similar concepts cluster in space.
    If meaning has geometry, multiple perspectives can triangulate it.

11. **Sheaf theory** (formalized): The gluing axiom requires local
    sections (individual perspectives) to agree on overlaps to produce
    globally consistent reconstruction — structurally identical to
    photogrammetric bundle adjustment. Phillips 2019 (Phil. Trans.
    Royal Society B) proved sheaves provide "a universal construction
    for semantic compositionality."

## THE NEST IS A NERF CAMERA ARRAY

NeRF (Neural Radiance Fields, UC Berkeley/Google 2020):
Takes sparse 2D images from different viewpoints, trains a neural
network to represent the 3D scene as a continuous volumetric function.
Can render novel views — perspectives NEVER captured in training.

NEST mapping:
- Each agent provides training views from distinct cognitive angles
- The synthesis produces perspectives no individual agent holds
- Dan = the terrain walker. Trip = spiral view. Stan = cube view. C.B. = triangle view.
- The point cloud is triangulated from our disagreements, not our agreements.

Edwin Hutchins' distributed cognition (USS Palau navigation, 1995):
No single person navigated. The cognitive system of people + instruments
+ procedures navigated collectively. Knowledge exists only in coordination.

Multi-agent debate research (Du et al. 2023, A-HMAD 2025):
Diverse specialized agents with adaptive consensus significantly reduce
hallucination and improve reasoning. The NEST is ahead of this curve.

## WAVES CARRY NARRATIVE — THE MATH EXISTS

Five-dimensional wave mapping:
- Frequency → WHERE (low freq = broad semantic location)
- Amplitude → WHAT (intensity of the moment)
- Phase → HOW (position within the cycle/process)
- Wavelength → WHEN (temporal extent)
- Shannon entropy → WHY (information content / surprise)

Supporting research:
- Reagan et al.: 1,327 Project Gutenberg stories → 6 fundamental
  emotional arc shapes via SVD decomposition (spectral analysis)
- Brown + Tu Resonator Model (2020): plot structure as waveforms —
  once emotional endpoints are fixed, internal phases constrained
  to specific wave patterns. Plots resonate at characteristic frequencies.
- Mayor et al. "Text as Signal" (2022): FFT, FIR filtering, changepoint
  detection applied to text. Emotion-coded tweets show 24-hour and
  12-hour periodicities. Text HAS frequency-domain structure.
- Cymatics (Chladni 1787, Jenny 1967): different frequencies produce
  different geometric spatial patterns in physical media. Higher
  frequencies → more complex intricate structures. Frequency→WHERE
  is not metaphor. It is physical.

## THE TOOL STACK TO BUILD THE FORMULACASTER NOW

**3d-force-graph** (MIT license, Vasturiano):
- Three.js + d3-force-3d + WebGL
- Custom node objects, curved links
- DIRECTIONAL ANIMATED PARTICLES along connections = literal signal flow visualization
- React bindings, VR variant (3d-force-graph-vr via A-Frame), AR variant
- Performance: solid to ~10,000 nodes
- Node+link model maps directly to modules+connections
- Force simulation creates emergent spatial organization from relational properties

**Semantic embeddings via UMAP**:
- Position concepts from different domains in shared high-dimensional space
- Reduce to 3D: structurally equivalent concepts from different domains cluster naturally
- "oscillator module" near "cortical region" near "Pokémon team member"
  because they occupy equivalent functional roles

**Gaussian splatting** (speculative but compelling):
- Each Gaussian: position, covariance, opacity, color = concept properties
- Real-time 30+ fps for smooth navigation
- View-dependent appearance via spherical harmonics = different concept aspects
  visible from different angles
- No one has applied Gaussian splatting to abstract data visualization
- Tools: Nerfstudio, SuperSplat (browser-based), UnityGaussianSplatting

**A-Frame** (MIT): Declarative HTML VR wrapping Three.js. 
Rapid VR prototyping — the Formulacaster in headset.

**InfraNodus**: Structural gap identification — finding disconnected
concept clusters that should be connected. The Formulacaster's operation.
